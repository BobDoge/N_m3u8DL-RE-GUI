# N_m3u8DL-RE GUI 批量下载器

这是一个基于 Python `tkinter` 开发的图形界面工具，旨在简化命令行下载工具 [N_m3u8DL-RE](https://github.com/nilaoda/N_m3u8DL-RE) 的使用。它支持单个链接下载和基于 CSV 文件的批量下载任务，并提供实时的进度监控。

## ✨ 主要功能

* **双模式输入**：支持手动输入单个 M3U8/MPD 链接，或通过 CSV 文件导入成百上千个下载任务。
* **实时进度显示**：通过底层正则匹配，将命令行繁杂的进度信息提取并显示在专门的进度状态栏中，保持日志区整洁。
* **自定义配置**：支持设置保存目录、自定义文件名以及多线程并行下载（`--thread-count`）。
* **异步执行**：下载任务在后台线程运行，界面不会因下载过程而卡死。

## 🛠️ 环境要求

1. **Python 3.x**: 确保已安装 Python 环境。
2. **N_m3u8DL-RE 核心程序**:
* 你需要先从 [N_m3u8DL-RE Releases](https://github.com/nilaoda/N_m3u8DL-RE/releases) 下载对应系统的可执行文件。
* **重要**：请将下载好的 `N_m3u8DL-RE` 可执行文件放入系统的 **PATH 环境变量** 中，或者放在与此脚本相同的目录下。


3. **FFmpeg**: 建议安装 FFmpeg 以进行视频合并和格式转换。

## 🚀 使用说明

### 1. 运行程序

在终端或命令行中运行：

```bash
python N_m3u8DL-RE_GUI.py
```

### 2. 下载模式

* **单个链接**：在“单个链接”输入框粘贴地址，点击“开始执行”。
* **批量下载**：
1. 准备一个 `.csv` 文件。
2. 格式要求：**第一列为视频链接，第二列为保存的文件名**（不含后缀）。
3. 点击“浏览...”选择该文件，然后点击“开始执行”。



### 3. 参数设置

* **保存目录**：默认为当前脚本目录，建议手动选择一个存储空间充足的文件夹。
* **线程数**：默认 16 线程，可根据你的带宽情况调高或调低。

## 📂 CSV 格式示例

你可以使用 Excel 或记事本创建一个如下格式的文件：

```csv
https://example.com/video1.m3u8,第一课_导学
https://example.com/video2.m3u8,第二课_基础知识
```

## 📝 注意事项

* **编码问题**：脚本默认使用 `gbk` 编码读取命令行输出（适配 Windows 控制台）。如果你在 Linux/macOS 上使用，可能需要将代码中的 `encoding='gbk'` 修改为 `encoding='utf-8'`。
* **权限**：请确保程序有权在你选择的保存目录中写入文件。
